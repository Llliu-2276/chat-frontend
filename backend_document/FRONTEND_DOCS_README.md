## 📊 项目概览

### 已实现功能模块

- ✅ **用户认证**：注册、登录、登出、Token管理
- ✅ **用户资料**：查看、修改用户名、修改密码
- ✅ **在线状态**：心跳机制、在线人数统计
- ✅ **用户搜索**：用户名/账号搜索、分页查询
- ✅ **好友系统**：好友列表、未读消息、聊天记录、好友申请（双向确认）、删除好友（含聊天记录清理）
- ✅ **群组系统**：创建群聊、群列表、群详情、解散/退出、发送消息、聊天记录、成员列表、WebSocket 实时群聊+变动通知、**踢人出群**、**群主转让**、**邀请好友入群**、**撤回群消息**、**消息搜索**
- ✅ **WebSocket**：私聊/群聊消息推送、上线/离线通知、已读回执、好友申请通知、群成员变动通知、**消息撤回通知**、**群主转让通知**
- ✅ **用户拉黑**：拉黑/取消拉黑、黑名单列表、发消息/好友申请拦截
- ✅ **接口限流**：Redis 滑动窗口限流，默认 60次/60秒

### 计划中功能模块

后续可扩展：文件/图片上传、Swagger 集成、正在输入状态等（详见 TODO_NEXT.md）

## 💡 开发建议

### 1. 使用测试工具
- **Postman**：手动测试接口
- **浏览器开发者工具**：查看网络请求
- **PowerShell脚本**：自动化测试

### 2. 参考代码示例
文档中提供了大量的代码示例，可以直接复制使用：
- JavaScript原生fetch
- Vue.js (axios)
- React (fetch封装)

### 3. 注意认证流程
```
登录 → 保存Token → 启动心跳 → 携带Token请求 → 登出清除Token
```

这是最核心的流程，务必理解透彻。

## 📞 获取帮助

### 遇到问题时
1. **查阅文档**：90%的问题文档中都有答案
2. **检查日志**：浏览器控制台 + 后端日志
3. **使用工具**：Postman验证接口是否正常

### 常见问题
- ❓ 接口401错误 → Token未正确携带或已过期
- ❓ 跨域错误 → 后端已配置CORS，检查请求地址
- ❓ 数据为空 → 检查数据库是否有数据
- ❓ 心跳失败 → 检查Token是否有效
